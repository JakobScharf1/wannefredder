Setup Flask

Introduction

This approach allows you to run a Flask project on a client system without setting up the entire project structure. It assumes you have access to the project's requirements.txt file, which lists all required Python packages.

Prerequisites

Please follow below steps to install and setup all prerequisites:

  • Python

    Make sure to have the Python installed & running in your computer. If you already have installed Python on your computer, you can skip this step .

  • Pip

    Run pip --version in your terminal. If pip is not installed, it usually comes bundled with Python. Download Python again if necessary.

  • Git

    Make sure to have the Git installed globally & running on your computer. If you already have installed git on your computer, you can skip this step.

Installation

To setup the template or theme, follow below-mentioned steps:

  • Install Prerequisites

    Make sure to have all above prerequisites installed & running on your computer

After you finished with the above steps, you can run the following commands into the terminal / command prompt from the root directory of the project to run the project locally or build for production use:

  • Navigate to the project directory:

    Use the cd command to navigate to the directory containing the unzipped project files.

  • Install required packages (within the virtual environment):

    Bash
    Pip install flask if not install flask.
    This installs the necessary packages for your project inside the virtual environment.

Freezing Dependencies (Optional):
  • Freezing Dependencies (Optional):

    You can create a snapshot of the currently installed packages and their versions using:

    Bash
    pip freeze
    This will update your requirements.txt file with the exact versions of the installed packages. This can be useful for ensuring reproducibility or sharing your project's dependencies in the future.

Running the Server
  • Navigate to the project directory:

    Assuming you have the project directory containing manage.py, navigate to it using cd.

  • Run development server:

    Execute the following command:

    Bash
    python app.py
    This can be useful for ensuring reproducibility or sharing your project's dependencies in the future.

  • This starts the Flask development server, usually accessible at http://localhost:8000/ by default.
