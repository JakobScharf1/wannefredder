Prerequisites for Setting Up the Project

Please follow below steps to install and setup all prerequisites:

  • PHP >= 8.x

  • Composer

    Make sure you have the Composer installed & running on your computer. If you already have installed Composer on your computer, you can skip this step. Download Composer

  • Node.js

    In order to use build tools you will need to download and install Node.js. If you do not have Node.js installed already, you can get it by downloading the package installer from the official website. Please download the stable version of Node.js (LTS). Download Node.js

Contact Form Configuration:

  • For app_local.php File

    Update the following lines in the app_local.php file to configure email sending:

Remove this line

'EmailTransport' => [
   'default' => [
      'host' => 'localhost',
      'port' => 25,
      'username' => null,
      'password' => null,
      'client' => null,
      'url' => env('EMAIL_TRANSPORT_DEFAULT_URL', null),
   ],
],

Add this line

'EmailTransport' => [
   'default' => [
      'className' => 'Smtp',
      'host' => 'smtp.gmail.com',
      'port' => 587,
      'username' => 'your-email@gmail.com', # Replace with your email
      'password' => '', # Replace with the app-specific password
      'tls' => true,
      'client' => null,
      'url' => env('EMAIL_TRANSPORT_DEFAULT_URL', null),
   ],
],

Key Points:

  • 1. username

  • the email address you’re using to send the emails.

  • 2. password:

  • This should be the App Password generated for your email account, not your actual email password.

  • For Gmail, app passwords can be generated from your Google Account if 2-Step Verification is enabled.

  • Refer to this Google support link to create an app password.
