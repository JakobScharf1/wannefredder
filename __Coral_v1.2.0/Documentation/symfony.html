Prerequisites for Setting Up the Project

Please follow below steps to install and setup all prerequisites:

  • PHP >= 8.x

  • Composer

    Make sure you have the Composer installed & running on your computer. If you already have installed Composer on your computer, you can skip this step. Download Composer

  • Symfony CLI

    The Symfony CLI is a developer tool to help you build, run, and manage your Symfony applications directly from your terminal. If you already have installed Composer on your computer, you can skip this step.

    Download Symfony CLI

CSS

Files are explained below:

File Description
style.css Coral uses the Bootstrap v5.3.7. The core style.css file is being used in all the pages.

Javascript

Files are explained below:

File Description
app.js This is a main js file. It contains the custom JS code needed for features including layout, slider, etc.
easy_background.js This is a js file. It contains the custom JS code needed for features including background_image, etc.
plugins.init.js This is a js file. It contains the custom JS code needed for features including Tobii lightbox, Data Counter, Typed Text animation, Back Button etc.
